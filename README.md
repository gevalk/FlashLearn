# FlashLearn

A simple flashcard web application built with Python and Flask.

## Features

- Random flashcards
- Show and hide answers
- Clean and responsive interface
- Flashcards loaded from a JSON file

## Technologies

- Python
- Flask
- Jinja2
- HTML
- CSS
- JavaScript
- JSON
- Git
- Github
